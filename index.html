Prework Study Guide

HTML

  • The head element contains information about the webpage, inc. metadata not visible to the user.
  • The body element represents the visible content shown to the user.
  • Order matters eg. h1, h2, h3...
  • 'src' is used for the location of the file.
  • 'alt' is used for appearance and functionality.
  • Every HTML element is essentially a box.
    • Each box consists of Margins outside the element, and the elements borders, padding and content.
  • link creates the connection to the external file.
  • rel attribute specifies the relationship between the current document and the linked document.
  • href attribute specifies the location (URL) of the external resource. href attribute is using a relative path. This means the location is derived from the current location (in this case index.html)

CSS

  • Padding property adds space around the content inside the element.
    • When you increase padding, you are actually making the entire element bigger!
    • Padding is transparent but is located inside the element.
  • Margin is the transparent buffer outside the element.
  • Combining selectors into a single rule with a comma allows the declaration to affect both selectors.
    • This helps write Do Not Repeat Yourself (DRY) code.
